# Movies Flix

A Web based movie browsing app (Netflix Clone). 🎥

## Tech Stack 🥪
### Frontend 👁‍🗨
- React.js ✨
- Material UI 🌟
- Sass 🟣

### API 🔗
- TMDB 🎬

## Features 🚀
- Browse Movies / TV Shows 🎬
- Trending Movies/ TV Shows 📈
- Popular Movies/ TV Shows 🥤
- Rates ⭐
- Search 🔍
- Watch Trailers 📷
- Cast 🤵
